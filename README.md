# odin_landing-page
This is the CSS section project in "The Odin Project" foundations course.
The goal of this project is to match a website layout given by The Odin Project, with additional styling
details for font size, colors, font-family, etc. While the layout had general, filler text since the project
is more focused on CSS, I decided to go a step further and add context to the project. Instead of filler, 
the webpage was themed around a sort of invasive species awareness homepage, which introduced some images
of invasive species and if you wanted to learn more, you could in theory, visit other pages or sign up to
a newsletter. While it may appear you can go elsewhere, the buttons and links are either non functional
or are linked to itself.

Some things I learned or considered while working on this:

I separated different sections I was working with so that I could differentiate it quickly from other sections. Same with the
CSS sheet. Maybe there is some other way to do this, posibly just collapsing the group so that it doesn't show up.

There are global styling choices I need to make. I could globally remove underlines from anchor/link elements and remove bullets list items.

With lists in mind, there are also default padding/margin stylings that come with lists and even the browser that I needed to consider.

I should style the elements in each section initially, that way when I work to position it, I can do so in the desired styling. When I first started, I was flexboxing first, then changing font size, weight, etc. This is bad because my positioning would be affected by all the element styling. Overall, style html content first, then their positioning on the page.

To help with flexbox visualizations, I made a CSS section for border display, making it easier to understand what the boxes were around the elements I worked with and how they were spaced from each other.

In the end, I think I did pretty well to match the example layout and even built the webpage around an invasive species theme WITH pictures taken by me.
